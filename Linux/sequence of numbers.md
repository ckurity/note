### seq (1) - print a sequence of numbers
```
$ whatis seq    
seq (1)              - print a sequence of numbers
```
### New line with `seq 1 3`
```
$ seq 1 3
1
2
3
```
### 1 line with `echo {1..3}`
```
$ echo {1..3}
1 2 3
```

### Floating numbers
```
$ seq 0.1 0.3    
0.1

$ seq 0.1 0.1 0.3
0.1
0.2
0.3
```
